本地运行指南

由于网络限制，建议在本地运行完整版本：

方法1：运行Streamlit版本
python run_dashboard.py

然后访问：http://localhost:8501

方法2：运行Flask版本
python web/app.py

然后访问：http://localhost:5000

配置自定义域名

如果你有自己的域名，可以配置自定义域名来解决访问问题：

  1. 将域名添加到Cloudflare
  2. 在Workers设置中添加自定义域名
  3. 更新代码中的API地址

详细步骤请参考：自定义域名配置指南
